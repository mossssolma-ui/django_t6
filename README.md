# Django 6 - Интернет-магазин

Учебный проект интернет-магазина на Django.

## Описание проекта

Проект представляет собой интернет-магазин, который будет постепенно дорабатываться в течение всего курса.
На данный момент реализована базовая структура проекта с двумя страницами.

## Функциональность

- **Главная страница** (`/`) - приветственная страница с информацией о магазине
- **Страница контактов** (`/contacts/`) - страница с контактной информацией и формой обратной связи
- **Форма обратной связи** - отправка сообщений через форму на странице контактов

## Технологии

- Python 3.14+
- Django 6.0.2
- psycopg2 2.9.11
- python-dotenv 1.2.2
- ipython 9.10.0
- pillow 12.1.1
- Poetry - управление зависимостями
- Bootstrap 5 - стилизация

## Установка и запуск

### Предварительные требования

- Python 3.14 или выше
- Poetry (менеджер зависимостей)

### Установка

1. Клонировать репозиторий:
    ```bash
    git clone git@github.com:mossssolma-ui/django_t6.git
    cd django-6

2. Установить зависимости Poetry
    ```bash
    poetry install

3. Запустить сервер разработки
   ```bash
   python manage.py runserver

4. Открыть в браузере:
[http://127.0.0.1:8000/](http://127.0.0.1:8000/) - главная страница
[http://127.0.0.1:8000/contacts/](http://127.0.0.1:8000/contacts/) - страница контактов
